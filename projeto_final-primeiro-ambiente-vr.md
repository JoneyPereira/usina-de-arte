# Descrição Geral

Criar uma experiência simples em Realidade Virtual utilizando Unity e, caso o aluno deseje, Meta SDK, aplicando todos os conceitos básicos aprendidos durante a primeira fase do curso.

## Objetivo

Demonstrar compreensão dos fundamentos de XR através da criação de um ambiente navegável simples.

## Requisitos do Projeto

1. Configuração Técnica

- Projeto Unity criado com versão compatível com Meta SDK
- Meta XR SDK corretamente instalado e configurado
- Build Settings preparados para plataforma Android (Meta Quest)
- XR Plugin Management configurado adequadamente
- A movimentação inicial deve ser toda feita no PC, sem necessitar de funcionar apenas com o óculos de VR.

2. Ambiente Virtual

- Mínimo de 5 objetos 3D na cena (podem ser primitivos ou assets importados)
- Um plano de chão ou terreno onde o usuário possa "caminhar"
- Skybox configurado (céu/ambiente ao redor)
- Objetos posicionados de forma coerente, criando um ambiente reconhecível

3. Organização e Qualidade

- Hierarquia de objetos organizada com pastas lógicas
- Nomenclatura clara e consistente de todos os elementos
- Projeto limpo, sem arquivos desnecessários
- README.txt com instruções básicas

## Entregas Necessárias

1. Arquivo do Projeto

- Projeto Unity completo em repositório no GitHub – anexe um documento .txt com o link do repositório.
- Incluir apenas as pastas necessárias (Assets, ProjectSettings, Packages)
